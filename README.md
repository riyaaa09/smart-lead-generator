# Smart Lead Generator 🚀

## Overview
This project is a simple lead generation tool that automates the process of finding company websites and extracting emails.

## Features
- Search-based lead generation
- Email extraction from websites
- Lead scoring (High/Low)
- CSV download

## Tech Stack
Python, Streamlit, BeautifulSoup, Pandas

## How to Run
pip install -r requirements.txt  
streamlit run app.py

## Business Value
This tool reduces manual effort in lead generation and helps businesses focus on high-quality prospects.
